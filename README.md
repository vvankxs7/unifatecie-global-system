# 🌍 Unifatecie Global System
> **Status:** 🛠️ Em desenvolvimento (Fase 1: Lógica de Negócio & POO)

Sistema internacional para gerenciamento de cursos da Unifatecie, desenvolvido com foco em **Software Architecture** e **DevSecOps**.

## 🚀 Roadmap de Carreira & Projeto
- [x] 🏗️ Estruturação de Repositório Corporativo.
- [ ] ☕ **Fase 1 (Atual):** Modelagem POO em Java (Classes `Curso` e `Lead`).
- [ ] 🌐 **Fase 2:** Interface Web (HTML5/CSS3/TypeScript).
- [ ] 🗄️ **Fase 3:** Persistência de dados com PostgreSQL.
- [ ] 🛡️ **Fase 4:** Auditoria de Segurança e Hardening (OWASP).

## 🛡️ Implementações de Segurança (Cybersecurity DNA)
Como foco em Red Team e DevSecOps, este projeto aplica:
* **Encapsulamento Estrito:** Atributos privados e métodos de acesso controlados.
* **Sanitização de Inputs:** Proteção contra ataques de injeção logo na camada de domínio.
* **Data Masking:** Tratamento de dados sensíveis conforme a LGPD.
